Ejercicio práctico:
    Este es un formulario de login y/o registro de usuario.
    Ahora mismo no está implementado su comportamiento. 

    Vista por defecto:
      Cuando un usuario accede por primera vez, debe mostrarse una pantalla con el siguiente formato:
        Titulo: login
        Campos: Email y Password
        Link a Register
        Botón Submit.
      
    Acción Register:
      Se accede a ella por medio del link de "Register"
      Los campos a mostrar son:
        Titulo Register.
        Campos: todos.
        Link a login
        Boton Submit.

    Accion Login:
      Se visualizará la vista "por defecto".
      Si el login es correcto, mostrar un span con el mensaje "Usuario correcto".
      Si el login es incorrecto, debe mostrar un span "Usuario incorrecto".

    Estilos:
      Modifica los estilos el formulario de registro de usuario para que los elementos queden alineados.

    Extras:
      Sabrias crear un sistema de validaciones de campos? 
        Ejemplo: si el email o pass es vacío mostrar su mensaje.

      Sabrías hacer responsive el formulario?

      ¿Sabrias proponer mejoras de implementar esta funcionalidad?
